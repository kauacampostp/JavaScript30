# JavaScript30

JavaScript30 is a free challenge available at [courses.wesbos.com](https://courses.wesbos.com/).

This course consists of 30 JavaScript projects designed to help developers master the DOM and build logic without the help of libraries or frameworks.

## 🎯 My Objective
My goal is to **consistently practice** building mini-projects to strengthen my core web development skills. I am focusing on mastering the fundamentals before moving deeper into modern frameworks.

## 🛠️ Built With
* **HTML5** (Semantic tags)
* **CSS3** (Transitions and Flexbox)
* **Vanilla JavaScript** (DOM Manipulation and Events)
